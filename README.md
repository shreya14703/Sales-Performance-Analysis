# 📊 Sales Performance Dashboard – Excel Project

## 📌 Project Overview
This project demonstrates an interactive **Sales Performance Dashboard** built in **Microsoft Excel**. The dashboard provides a comprehensive analysis of sales, profit, discounts, and regional performance across various product categories.  

The goal of this project is to convert raw sales data into meaningful business insights using Excel features such as **Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting**.

---

## 🎯 Objectives
- Analyze total sales, total profit, and average discount trends.  
- Evaluate sales performance by **Category, Subcategory, Region, and Segment**.  
- Visualize profit distribution and discount patterns for informed decision-making.  
- Enable dynamic filtering using **Slicers** for interactive analysis.  
- Provide insights for improving revenue, discount strategy, and regional targeting.

---

## 📁 Dataset Description
The dashboard is built on a dataset containing transactional sales data, including:

| Column Name       | Description                                      |
|------------------|--------------------------------------------------|
| Order Date        | Date of the order                                |
| Category          | Product category (Furniture, Office Supplies, Technology) |
| Subcategory       | Specific product type                            |
| Sales             | Revenue generated by the order                   |
| Profit            | Profit earned from the order                     |
| Discount          | Discount applied to the order                    |
| Region            | Region where the order was placed (East, West, South, Central) |
| Segment           | Customer segment (Consumer, Corporate, Home Office) |
| Quantity          | Number of units sold                             |

---

## 📊 Dashboard Features

### 1. KPI Metrics
- **Total Sales** – Shows the total revenue generated.  
- **Total Profit** – Displays cumulative profit.  
- **Average Discount** – Highlights the average discount applied across all orders.

### 2. Sales & Profit Analysis
- **Sales by Category** – Bar chart comparing total sales across product categories.  
- **Profit by Category** – Visualizes profit distribution for each category.  
- **Average Discount by Category** – Shows how discounts vary across categories.

### 3. Advanced Analytics
- **Sales by Region** – Regional sales performance for better geographical targeting.  
- **Sales by Segment** – Identifies top-performing customer segments.  
- **Sales by Subcategory** – Detailed breakdown of subcategory performance.  
- **Sales Distribution by Quantity** – Understand the distribution of order quantities.

### 4. Interactive Filters (Slicers)
Users can dynamically filter data by:
- **Order Date**  
- **Category**  
- **Region**  
- **Segment**  

This allows for real-time exploration of trends and metrics.

## 🔍 Key Insights
- The **Furniture category** has the highest sales and profit contribution.  
- Discounts vary significantly by category; strategic adjustments can improve profit margins.  
- The **West and South regions** contribute the most to overall sales.  
- Consumer and Corporate segments are the largest revenue generators.  

---

## 💡 Business Recommendations
- Focus promotions and inventory on **high-performing categories** like Furniture.  
- Adjust discount strategies for categories with lower profit margins.  
- Target **top regions and segments** with marketing campaigns.  
- Monitor sales trends regularly to identify seasonal peaks.  

---

## 🚀 How to Use
1. Open the Excel file in Microsoft Excel (2016 or later recommended).  
2. Navigate to the **Dashboard** sheet.  
3. Use the **Slicers** on the right-hand side to filter by date, category, region, or segment.  
4. Observe how KPIs, charts, and metrics update dynamically based on your selections.  
5. Explore all visualizations to derive insights on sales, profit, and discount patterns.
